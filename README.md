# Ludo Console Game

A console-based implementation of the Ludo board game in MASM32 assembly language.

## Features

- 2-4 players
- Turn-based gameplay
- Full standard Ludo rules
- Console-based interface with colors

## Building

Run `build.bat` to assemble and link the program.

## Running

After building, run `ludo.exe` from the command prompt.

## Requirements

- MASM32 assembler
- Windows console

## Game Rules

Standard Ludo rules apply. Roll dice to move pieces from home to the center.
